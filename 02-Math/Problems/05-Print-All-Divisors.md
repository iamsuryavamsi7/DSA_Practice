# Problem 05 - Print All Divisors

## Problem Statement

Given an integer `n`, print all of its divisors.

A divisor is a number that divides the given number completely without leaving any remainder.

---

# Examples

## Example 1

Input

```text
12
```

Output

```text
1 2 3 4 6 12
```

---

## Example 2

Input

```text
15
```

Output

```text
1 3 5 15
```

---

## Example 3

Input

```text
7
```

Output

```text
1 7
```

---

# Constraints

- Input will be a positive integer.
- Assume the number fits within Java's integer range.

---

# Expected Output

Print all divisors of the given number.
