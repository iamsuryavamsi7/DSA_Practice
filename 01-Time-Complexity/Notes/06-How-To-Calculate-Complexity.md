# How to Calculate Time & Space Complexity

## Golden Rule

Whenever you see code, ask two questions:

1. How many times does the code execute?
   → Time Complexity

2. How much extra memory is created?
   → Space Complexity

---

## Common Patterns

### Simple Statement

Time: O(1)

Space: O(1)

---

### Single Loop

Time: O(n)

Space: O(1)

---

### Consecutive Loops

Time: O(n)

Space: O(1)

---

### Nested Loops

Time: O(n²)

Space: O(1)

---

### Extra Array

Time: O(n)

Space: O(n)

---

### Matrix

Time: O(n²)

Space: O(n²)

---

## Important

Input arrays are NOT counted in Space Complexity.

Only additional memory created by the algorithm is counted.
