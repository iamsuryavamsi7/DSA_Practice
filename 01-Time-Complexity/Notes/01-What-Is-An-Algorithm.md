# What is an Algorithm?

## Definition

An algorithm is a finite sequence of well-defined instructions used to solve a problem.

## Characteristics

- Input
- Output
- Finite
- Well Defined
- Effective

## Examples

- Google Maps
- YouTube Recommendation
- ATM
- Recipe
- Searching
- Sorting

## Important

An algorithm is independent of the programming language.

Java, Python and C++ are only implementation languages.

## Key Learning

A problem can have multiple algorithms.

The goal of DSA is to find the most efficient one.
