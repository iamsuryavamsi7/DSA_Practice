# What is Time Complexity?

## Definition

Time Complexity measures how the running time of an algorithm grows as the input size grows.

## Important Points

- It does NOT measure actual seconds.
- It measures growth.
- It helps compare algorithms.
- It ignores hardware differences.

## Why is it Important?

As input grows:

- Slow algorithms become unusable.
- Efficient algorithms continue to perform well.

## Key Idea

Time Complexity focuses on how the amount of work changes as the input size increases.

## Example

Input:

10 → 100 → 1000

Algorithm A:

10 → 100 → 1000 operations

Algorithm B:

100 → 10000 → 1000000 operations

Algorithm A scales much better.

## Key Learning

Time Complexity is about scalability, not execution time.
