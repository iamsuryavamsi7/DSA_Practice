# Move Zeroes

## Problem Statement

Given an integer array, move all the zeroes to the end of the array while maintaining the relative order of the non-zero elements.

The operation must be performed in-place.

---

## Learning Objectives

- Strengthen the Two Pointers technique.
- Learn stable in-place array modification.
- Understand element swapping.
- Preserve relative ordering.

---

## Topics Covered

- Arrays
- Two Pointers
- In-place Modification
- Stable Rearrangement

---

## Expected Complexities

Brute Force

Time Complexity

```
O(n²)
```

Space Complexity

```
O(n)
```

Optimal

Time Complexity

```
O(n)
```

Space Complexity

```
O(1)
```
