# Find Missing Number

## Problem Statement

Given an array containing `n - 1` distinct numbers from the range `1` to `n`, find the missing number.

---

## Example

Input

```text
[1,2,4,5]
```

Output

```text
3
```

---

## Learning Objectives

- Compare multiple approaches to solve the same problem.
- Learn mathematical optimization.
- Introduce XOR as a problem-solving technique.
- Understand time-space trade-offs.

---

## Topics Covered

- Arrays
- Searching
- Mathematics
- XOR

---

## Expected Approaches

### Brute Force

Check every number from `1` to `n`.

Time

```
O(n²)
```

Space

```
O(1)
```

---

### Better

Use a Hash Array / HashSet.

Time

```
O(n)
```

Space

```
O(n)
```

---

### Optimal

Use the Sum Formula.

Time

```
O(n)
```

Space

```
O(1)
```

---

### Optimal

Use XOR.

Time

```
O(n)
```

Space

```
O(1)
```
