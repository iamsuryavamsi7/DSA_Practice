# Union of Two Sorted Arrays

## Problem Statement

Given two sorted arrays, return the union of both arrays.

The union should contain only unique elements and must remain sorted.

---

## Example

Input

```text
Array 1 = [1,2,3,4,5]

Array 2 = [2,3,4,4,5,6]
```

Output

```text
[1,2,3,4,5,6]
```

---

## Learning Objectives

- Learn to traverse two arrays simultaneously.
- Understand duplicate removal during merging.
- Strengthen the Two Pointers technique.

---

## Topics Covered

- Arrays
- Two Pointers
- Merging
- Duplicate Removal

---

## Expected Approaches

### Brute Force

Store all elements in a Set.

Time

```
O((n+m) log(n+m))
```

Space

```
O(n+m)
```

---

### Optimal

Use Two Pointers.

Time

```
O(n+m)
```

Space

```
O(n+m)
```
