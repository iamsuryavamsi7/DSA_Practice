# Intersection of Two Sorted Arrays

## Problem Statement

Given two sorted arrays, return their intersection.

The intersection should contain only the elements that are present in both arrays.

Duplicate elements should appear only once in the result.

---

## Example

Input

```text
Array A = [1,2,2,3,4,5]

Array B = [2,2,3,5,6]
```

Output

```text
[2,3,5]
```

---

## Learning Objectives

- Strengthen the Two Pointers technique.
- Compare Union vs Intersection.
- Learn synchronized traversal of two sorted arrays.
- Handle duplicates efficiently.

---

## Topics Covered

- Arrays
- Two Pointers
- Merging
- Duplicate Handling

---

## Expected Approach

### Optimal

Use Two Pointers.

Time

```
O(n + m)
```

Space

```
O(min(n,m))
```

(The output list stores only common elements.)
