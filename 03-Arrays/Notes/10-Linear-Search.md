# Linear Search

## Problem Statement

Given an array and a target value, find the index of the target element.

Return the index if found; otherwise return `-1`.

---

## Learning Objectives

- Understand sequential searching.
- Learn when linear search is appropriate.
- Compare searching techniques.

---

## Topics Covered

- Arrays
- Traversal
- Searching

---

## Expected Complexities

Time Complexity

```
O(n)
```

Space Complexity

```
O(1)
```
