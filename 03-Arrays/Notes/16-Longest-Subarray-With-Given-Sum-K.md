# Longest Subarray With Given Sum K

## Problem Statement

Given an integer array and an integer K, find the length of the longest subarray whose sum is exactly K.

---

## Example

Input

```text
Array = [1,2,3,1,1,1,1]

K = 3
```

Output

```text
2
```

Explanation

The longest subarrays are

```
[1,2]

or

[3]
```

The maximum length is 2.
