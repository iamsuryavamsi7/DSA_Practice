# Problem 02 - Reverse Number

## Problem Statement

Given an integer `n`, reverse its digits and return the reversed number.

---

# Examples

## Example 1

Input

```text
12345
```

Output

```text
54321
```

---

## Example 2

Input

```text
9876
```

Output

```text
6789
```

---

## Example 3

Input

```text
9070
```

Output

```text
709
```

---

## Constraints

- Input will be a non-negative integer.
- Assume the number fits within Java's integer range.

---

# Expected Output

Return the reversed number.
