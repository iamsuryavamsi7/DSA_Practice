# Problem 09 - Prime Factorization

## Problem Statement

Given a positive integer `n`, return all of its prime factors.

A prime factorization expresses a number as the product of only prime numbers.

---

# Examples

## Example 1

Input

```text
n = 60
```

Output

```text
2 2 3 5
```

Explanation

```text
60

↓

2 × 2 × 3 × 5
```

---

## Example 2

Input

```text
n = 84
```

Output

```text
2 2 3 7
```

---

## Example 3

Input

```text
n = 97
```

Output

```text
97
```

Because 97 is already a prime number.

---

# Constraints

- n > 1

---

# Expected Output

Return all prime factors of the given number.
