# Problem 01 - Count Digits

## Problem Statement

Given an integer `n`, count the total number of digits present in the number.

---

## Examples

### Example 1

Input

```text
12345
```

Output

```text
5
```

---

### Example 2

Input

```text
7
```

Output

```text
1
```

---

### Example 3

Input

```text
987654321
```

Output

```text
9
```

---

## Constraints

- Input will be a non-negative integer.
- Assume the number fits within Java's integer range.

---

## Expected Output

Return the total number of digits present in the given number.

---

## Approach

Do not directly jump into coding.

Follow the learning workflow:

1. Understand the problem.
2. Think of your own solution.
3. Build the brute-force algorithm.
4. Analyze Time Complexity.
5. Analyze Space Complexity.
6. Optimize if possible.
7. Dry Run.
8. Write Java code.
