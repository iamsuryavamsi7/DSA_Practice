# Problem 06 - Prime Number

## Problem Statement

Given a positive integer `n`, determine whether it is a Prime Number.

A Prime Number has exactly two divisors:

- 1
- Itself

---

# Examples

## Example 1

Input

```text
13
```

Output

```text
true
```

---

## Example 2

Input

```text
12
```

Output

```text
false
```

---

## Example 3

Input

```text
1
```

Output

```text
false
```

---

# Constraints

- Input will be a positive integer.

---

# Expected Output

Return

- true if the number is Prime.
- false otherwise.
