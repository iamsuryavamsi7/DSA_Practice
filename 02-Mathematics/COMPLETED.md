# Complexity Improvements

| Problem | Brute Force | Optimized |
|---------|------------|-----------|
| Divisors | O(n) | O(√n) |
| Prime Check | O(n) | O(√n) |
| GCD | O(min(a,b)) | O(log(min(a,b))) |
| LCM | O(max(a,b)) | O(log(min(a,b))) using GCD |
| Prime Factorization | O(n) | O(√n) |
| Sieve of Eratosthenes | O(n√n) | O(n log log n) |
| Binary Exponentiation | O(n) | O(log n) |
